Tools

This page lists some general purpose tools that I have built.

QuickStat

A small package that allows for quick running of Single and Multiple OLS regressions. The package even includes support for calculating heteroskedastic robust standard errors using White's method!

Implemented in Python using Numpy, Pandas, and Matplotlib

The Github repository contains the package source code (LinReg.py) as well as a sample CLS application (SampleApp.py).

To run a regression, include a desired CSV file in the project's root directory and run SampleApp.py!

Github

BulkTransfer

A working prototype of a bulk file transfer tool.

Implemented in Python with PyQT, the tool compares the hashes of each file's binary data in a source directory to a destination directory, and writes a copy to the source directory if no matching hash is found.
