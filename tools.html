Scrapers

This page lists some of the web scrapers that I have written. All of these scraper are written in Python 3.X.
All of these scrapers use BeautifulSoup to parse HTML text. When possible, the requests package is used to connect to a site and pull the raw HTML.
When a web page makes use of JavaScript-generated HTML, I usually make use of the Selenium web driver.

I hope to make a scraper infrastructure application soon. The application would allow users to easily manage scraper use, as well as standardize scraper outputs to data bases of the user's choice!

The AESO CSD Scraper is a Python application that pulls data from the AESO's (Alberta Electric System Operator) current supply/demand report.

Data is pulled with respect to asset tag and fuel type, and reports available capacity, generated capacity, etc.
